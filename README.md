# 📸 F10 SS Aracı v1.0

**Birden fazla monitörünüz varsa, istediğiniz monitörden istediğiniz tuşla arka planda ekran görüntüsü almanızı sağlayan hafif bir masaüstü uygulaması.**

Birden fazla ekranla çalışırken veya oyun oynarken pencere değiştirmeden, belirlediğiniz tek bir tuşla belirlediğiniz ekrandan anında görüntü almanızı sağlar.

---

## Özellikler

- **Çoklu Monitör Desteği** — Sistemdeki tüm monitörleri algılar ve spesifik seçim imkanı sunar.
- **Özelleştirilebilir Kısayol** — F9, F10, F11, F12, Print Screen, Insert, Home tuşları atanabilir.
- **Otomatik Adlandırma** — Sıralı dosya kaydı (`ss_001.png`, `ss_002.png`, …) ile dosyalarınız karışmaz.
- **Özel Kayıt Dizini** — Görüntülerin kaydedileceği klasörü kolayca seçebilirsiniz.
- **Sıfır Bağımlılık** — PyInstaller ile derlenmiş, kuruluma gerek duymadan doğrudan çalışan `.exe` versiyonu.
